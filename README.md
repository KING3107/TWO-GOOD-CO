TWO GOOD CO. – FRONTEND CLONE

--------------------------------------------------

PROJECT OVERVIEW

This project is a frontend recreation inspired by the official Two Good Co. website. 
The objective was not to achieve a pixel-perfect clone, but to closely replicate the 
overall user experience, including layout structure, smooth scrolling behavior, and 
interactive animations.

The project focuses on implementing modern frontend techniques and gaining hands-on 
experience with animation libraries and DOM manipulation.

--------------------------------------------------

KEY FEATURES

• Fully responsive design across devices  
• Smooth scrolling implementation  
• High-quality animations using GSAP  
• Interactive UI with JavaScript DOM manipulation  
• Clean and minimal layout inspired by the original design  
• Optimized structure for readability and scalability  

--------------------------------------------------

TECHNOLOGY STACK

• HTML5 – Semantic structure  
• CSS3 – Styling, layout, and responsiveness  
• JavaScript (ES6+) – Interactivity and logic  
• GSAP (GreenSock) – Animation handling  
• Locomotive Scroll – Smooth scrolling effects  

--------------------------------------------------

LEARNING OUTCOMES

• Practical understanding of DOM manipulation  
• Integration of third-party animation libraries  
• Creating smooth and engaging user experiences  
• Responsive design implementation  
• Structuring and organizing a real-world frontend project
